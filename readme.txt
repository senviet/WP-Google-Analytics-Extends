Show some Google Analytics\'s report in front-end by widget and shortcode

== Description ==
This plugin displays some google Analytics report information in the font-end. This is useful information for users:

*   The most Popular Posts
*   The most Popular Keyword
*   The pageview

Support : widget, shortcode

= This plugin have a tutorial on Sen Viet =
*   [GOOGLE ANALYTICS DASHBOARD EXTENDS: WIDGET BÀI VIẾT NỔI BẬT](http://laptrinh.senviet.org/wordpress-plugin/google-analytics-dashboard-extends-widget-bai-viet-noi-bat/)
*   [GOOGLE ANALYTICS DASHBOARD EXTENDS: SHORTCODE](http://laptrinh.senviet.org/wordpress-plugin/google-analytics-dashboard-extends-shortcode/)

Plugin này giúp hiển thị một vài thông tin report của google Analytics ra ngoài trang chủ. Đây là những thông tin có ích cho người dùng :

*   Bài viết phổ biến nhất
*   Keyword nổi bật
*   Số lượng khách truy cập.
